# timetohome

Usage:

```
$ timetohome 
Argument start is not set
  -apiKey string
        Developer API key
  -end string
        Coordinate of the ending point (like 49.238197,2.343180)
  -linkToVisit string
        Link to visit via 'Open MyDrive map' when clicked
  -start string
        Coordinate of the starting point (like 50.138197,2.273150)
```
